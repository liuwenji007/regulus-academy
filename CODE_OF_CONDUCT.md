# 贡献者行为准则

## 我们的承诺

为了营造一个开放和友好的社区，我们承诺尊重所有参与者——无论经验水平、性别、性取向、外貌、种族、宗教信仰或年龄。

## 我们的标准

### 鼓励的行为

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他参与者表示同理心

### 不可接受的行为

- 使用性暗示的语言或图像，以及不受欢迎的性关注或挑逗
- 挑衅、侮辱/贬低性评论，以及人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息，例如实际地址或电子地址
- 在专业环境中通常被认为不合适的其他行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并对任何不可接受的行为采取适当和公正的纠正措施。

维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、issue 和其他贡献，或暂时或永久禁止任何他们认为有不当、威胁、冒犯或有害行为的贡献者。

## 范围

本行为准则适用于项目空间和公共空间，当个人代表项目或其社区时。代表项目或社区的例子包括使用官方项目电子邮件地址、通过官方社交媒体账号发帖、或在在线或离线活动中担任指定代表。

## 执行

可以通过 GitHub Issues 或项目维护者的联系方式报告滥用、骚扰或其他不可接受的行为。所有投诉将被审查和调查，并将产生被认为必要且适合情况的回应。项目团队有义务对事件报告者保密。
